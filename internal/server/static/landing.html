🔍 VulnScanner
Dashboard Nuevo Escaneo API

Escanea vulnerabilidades web en segundos

VulnScanner es una herramienta CLI + API construida en Go que audita targets web en busca de vulnerabilidades comunes: puertos abiertos, cabeceras faltantes, SQLi, XSS, SSRF, LFI, y más. Un solo binario, sin dependencias externas.

▶ Iniciar escaneo

⚡ Rápido y concurrente

Worker pool con goroutines reales. Escanea múltiples módulos en paralelo con límite de rate configurable.

🛡️ 12 módulos de detección

Desde port scan hasta SSRF y subdomain enum. Cobertura OWASP básica lista para usar.

🔐 Escaneo autenticado

Login automático por formulario o token JSON. La sesión se inyecta en todas las peticiones del escaneo.

📊 Reportes múltiples

JSON, PDF, HTML, SARIF (GitHub Security) y Markdown. Elige el formato según tu pipeline.

🌐 API REST + UI

Integra en CI/CD vía API Gin o usa esta interfaz web embebida en el mismo binario.

📦 Sin CGO

Binario 100% portable, compilable para linux/darwin/windows en amd64/arm64. SQLite embebido sin CGO.

🚀 Uso rápido (CLI)

vulnscan scan example.com --full

vulnscan scan example.com --modules ssrf,lfi,redirect --workers 15

vulnscan scan example.com --auth-login-url URL -u user -p pass --format sarif

vulnscan serve --addr :8080 → levanta esta UI + API
